# Kotlin의 동일성 비교

## 내용의 동일성 비교
```
  - 서로 다른 메모리상에 할당된 객체여도 그 내용이 같은 경우, 동일하다고 판단
```

### 내용의 동일성을 판단하는 비교 연산자 : ==
```
    var a = 3
    var b = 3

    println(a == b)    
```
---

### 참고사항
 - '==' 연산자를 이용한 내용의 동일성 판단은 Kotlin의 모든 클래스가 내부적으로 상속받는 최상위 클래스인 'Any'의 equals() 함수가 반환하는 Boolean 값으로 판단

 - 기본 자료형에는 각 자료형의 특징에 따라 equals() 함수가 이미 구현되어 있지만, 개발자가 정의한 custom class에는 equals() 함수를 상속받아 동일성을 확인해 주는 함수를 별도로 구현해야함
    ```
        open fun equals(parmas: Any?): Boolean {
            ...
        }
    ```
---

## 객체의 동일성 비교
```
    - 서로 다른 변수가 메모리상의 같은 객체를 가리키는 경우, 객체가 동일하다고 판단
```
